weixin4j-qy
===========

[微信企业号](http://qydev.weixin.qq.com/wiki/index.php)开发工具包
---------------------------------------------------------------

功能列表
-------

  * AgentApi `应用设置API`

  * BatchApi `批量任务API`

  * ChatApi `会话服务API`

  * Helper `辅助API`

  * MediaApi `媒体素材API`
  
  * MenuApi `菜单管理API`
  
  * NotifyApi `消息发送API`

  * OauthApi `oauth授权登陆API`

  * PartyApi `部门管理API`

  * ProviderApi `服务商API`

  * SuiteApi `第三方应用API`

  * TagApi `标签管理API`
	
  * UserApi `成员管理API`
